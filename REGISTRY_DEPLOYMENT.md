# SolanaPilot On-Chain Registry Deployment Guide

## Overview

The SolanaPilot Registry is an Anchor-based Solana program that stores metadata about every program generated by SolanaPilot users, creating a public registry of AI-generated Solana programs.

## Prerequisites

- Rust 1.70+
- Anchor CLI 0.30.0
- Solana CLI 1.18+
- Node.js 18+

## Build the Program

```bash
cd programs/registry
anchor build
```

## Deploy to Devnet

```bash
# 1. Generate a keypair for the program (if you don't have one)
solana-keygen new -o registry-keypair.json

# 2. Deploy
solana program deploy \
  --url devnet \
  --keypair ~/.config/solana/id.json \
  target/deploy/solanapilot_registry.so \
  --program-id registry-keypair.json

# 3. Note your program ID and update in:
# - Anchor.toml
# - web-explorer/app/page.tsx
# - src/registryClient.ts
```

## Output Example

```
Program Id: <YOUR_PROGRAM_ID>
Signature: <TRANSACTION_SIGNATURE>
```

Update all config files with the actual program ID.

## Verify Deployment

```bash
solana program info <YOUR_PROGRAM_ID> --url devnet
```

## Instructions

### Register a Program

```typescript
await program.methods
  .registerProgram(
    programId, // PublicKey of the generated program
    programName, // string (max 50 chars)
    description, // string (max 200 chars)
    instructionCount, // u8 (number of instructions)
    generatorVersion, // string (SolanaPilot version, max 20 chars)
  )
  .accounts({
    programEntry: derivedPDA,
    creator: walletPublicKey,
    systemProgram: SystemProgram.programId,
  })
  .rpc();
```

### Log a Deployment

```typescript
await program.methods
  .logDeployment()
  .accounts({
    programEntry: derivedPDA,
  })
  .rpc();
```

## Account Structure

**ProgramEntry:**

- `program_id`: Pubkey (32 bytes)
- `program_name`: String[50] (54 bytes)
- `description`: String[200] (204 bytes)
- `instruction_count`: u8 (1 byte)
- `creator`: Pubkey (32 bytes)
- `timestamp`: i64 (8 bytes)
- `generator_version`: String[20] (24 bytes)
- `deployment_count`: u64 (8 bytes)
- `bump`: u8 (1 byte)

**Total space:** ~372 bytes
